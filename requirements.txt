# ToolRecall — zero external dependencies.
#
# The entire ToolRecall stack uses Python stdlib only:
#   - http.server    → HTTP proxy (toolrecall serve)
#   - sqlite3        → cache database (SQLite + FTS5)
#   - tomllib        → config parsing (Python 3.11+)
#   - hashlib        → cache key generation
#   - subprocess     → terminal command execution
#   - json           → serialization
#   - os, time, path → file operations
#
# Install: pip install toolrecall
# No pip install toolrecall[deps] needed — it just works.
